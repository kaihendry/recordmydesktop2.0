# recordmydesktop 2.0

https://s.natalian.org/2019-10-28/hardware-encoding.mp4

`x11capture` assumes **INTEL HARDWARE** for hardware encoding.

Relies on https://github.com/andykitchen/swcursor for showing a readable cursor.
